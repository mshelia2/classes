# Shelivan Banking Website

A premium static banking website prototype built with HTML and CSS. "Shelivan" is designed as a polished landing experience for modern digital banking, with a focus on security, account management, transfers, and user engagement.

## Overview

This project is a multi-page frontend demo for a banking brand called **Shelivan Premium Wealth**. It includes:

- A homepage hero section showcasing premium financial services
- A features page highlighting secure banking, transfers, analytics, and support
- A dashboard layout with balance summary, income, expenses, and recent transactions
- A transfer form for sending funds with a smooth UI
- A sign-in page and account creation flow mockup
- Client testimonials to build trust and credibility

## Pages

- `home.html` — Main landing page
- `features.html` — Product and service overview
- `dashboard.html` — Account dashboard interface
- `transfer.html` — Money transfer form
- `signin.html` — Sign-in page
- `get-started.html` — Account creation form
- `testimonials.html` — Customer feedback page
- `style.css` — Shared styling for the website

## Key Features

- Responsive navigation and layout
- Premium visual design with gold accents and modern typography
- Secure banking branding and trust elements
- Clear dashboard metrics and transaction list
- Simple, form-based user flows for sign in, onboarding, and transfers
- Testimonial cards for social proof

## Usage

1. Open the `first_project` folder in your browser or editor.
2. Open `home.html` to view the landing page.
3. Navigate between pages using the header links.
4. Edit the HTML and `style.css` to customize branding, text, and layout.

## Preview

- Open `home.html` directly in a browser to preview the site.
- For a local development server, use a tool such as VS Code Live Server or any simple static server.
- If using VS Code, install the Live Server extension and click `Go Live`.

## Notes

- This is a frontend prototype and does not include backend functionality.
- Forms are currently static and do not submit data.
- Google Fonts and Font Awesome are included via CDN in each page.
