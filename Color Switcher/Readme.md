# 🎨 Color Switcher & Mixer

A modern web application that changes the page background based on whatever color you type. Built with clean, simple code, this project lets you type individual color names or hex codes, or even type two colors together to watch them mix into a brand new color instantly!

---

## 🌟 Features

* **Type Any Color:** No boring buttons—just type any color name or hex code to change the background.
* **Smart Color Mixing:** Type two colors together (like `blue + yellow` or `red blue`) and the app automatically blends them.
* **Centered UI Design:** The main typing area stays centered in a clean white card with a soft drop shadow, while the background around it changes colors.
* **Smooth Transitions:** The background colors fade into each other smoothly instead of snapping instantly.

---

## 🛠️ Built With

* **HTML:** Creates the input field and text layout.
* **CSS:** Handles the layout centering, the box shadow, and the smooth color transitions.
* **JavaScript:** Listens to what you type, reads the colors, and calculates how to mix them.
