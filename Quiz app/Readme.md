# 🍎 Fruit Quiz App

A fun and interactive **Fruit Quiz Application** built with **HTML, CSS, and JavaScript**.  
The app allows users to register before playing, answer fruit-related questions, view their score, and review incorrect answers.

The project focuses on practicing JavaScript concepts such as **conditional statements, logical operators, DOM manipulation, arrays, functions, and event handling**.

---

## 🚀 Live Features

- 👤 User registration before starting the quiz
- 🍌 Fruit-based quiz questions
- 🔀 Randomized answer options
- ✅ Instant answer feedback
- 📊 Score calculation
- 📖 Review incorrect answers after completing the quiz
- 🔄 Replay quiz
- 👥 Switch to a new player
- 📱 Fully responsive design
- 🎨 Beautiful fruit background interface

# 🎮 How It Works

## 1. User Registration

Before playing, users must provide:

- Full name
- Email address
- Phone number

The quiz will only start when all required information is provided.

---

## 2. Answer Questions

Users answer fruit-related questions by selecting one option.

The app:

- Checks if the answer is correct
- Highlights the correct and wrong answers
- Keeps track of the player's score

---

## 3. View Results

After completing the quiz, users can see:

- Total score
- Performance message
- Incorrect answers with the correct answers

# 🧠 JavaScript Concepts Practiced

This project demonstrates:

## Conditional Statements

```javascript
if()
else if()
else
```

Used for checking answers and displaying results.

---

## Logical Operators

### AND Operator

```javascript
&&
```

Example:

```javascript
if(answer === correct && score > 0)
```

---

### OR Operator

```javascript
||
```

Used for combining multiple conditions.

---

## Truthy and Falsy Values

Example:

```javascript
if(!username)
```

Checks if a value is empty.

---

## Switch Statement

Used for handling different score outcomes.

Example:

```javascript
switch(score){

case 10:
console.log("Perfect");

break;

}
```

## DOM Manipulation

The app dynamically changes:

- Questions
- Answers
- Scores
- Results
- User information

Using:

```javascript
document.getElementById()
document.querySelectorAll()
```

## Arrays and Objects

Questions are stored as JavaScript objects:

```javascript
{

}
```
